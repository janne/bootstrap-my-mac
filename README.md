# bootstrap-my-mac

## Overview

An idempotent script to setup my Mac, based on:

- zsh
- homebrew
- neovim + [LazyVim](https://www.lazyvim.org/)
- asdf

## Setup

Prerequisites:

- A Mac with terminal access
- `git`

```
git clone https://github.com/janne/bootstrap-my-mac.git
cd bootstrap-my-mac
./bootstrap.sh
```

Feel free to delete `bootstrap-my-mac/` after running the bootstrap script.

## Fonts

## Local extensions

Extensions are added in `~/.config/`. See the folders `zsh/`, `nvim/` and `asdf/` in that folder for some examples.
